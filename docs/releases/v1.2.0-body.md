### Next release (maintainers)

1. Update **`CHANGELOG.md`** (move items from `[Unreleased]` to `## [X.Y.Z] - date`).
2. Create **`docs/releases/vX.Y.Z-body.md`** (copy structure from this file or the template).
3. Open/update GitHub Release:
   ```bash
   gh release create "vX.Y.Z" --title "vX.Y.Z" --notes-file docs/releases/vX.Y.Z-body.md
   ```
   For editing an existing release: `gh release edit "vX.Y.Z" --notes-file docs/releases/vX.Y.Z-body.md`  
   Full checklist: [RELEASE_NOTES_TEMPLATE.md](https://github.com/atezer/FMCP/blob/v1.2.0/docs/RELEASE_NOTES_TEMPLATE.md).
